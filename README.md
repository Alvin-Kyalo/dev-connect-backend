# dev-connect-backend

## How to contribute
Fork the repo. This will create a copy of the repo to your own repository list.<br>
Clone your forked repo to your project folder.<br>
<br>
Then:
```bash
cd dev-connect-backend

For windows run:
gradlew.bat build

For linux run:
./gradlew build
```

This installs the application dependencies needed. You are to use the wrapper to handle other tasks such as testing.
